* plac is general purpose tool than can be used to define
  command-oriented domain specific languages (DSLs). 

* it can be used as a command-line arguments parser with an
  easier-to-use API than argparse. 

* it can be used as a replacement of the cmd module in the standard 
  library.

* it can be used to automatize functional tests 

* and much more

This talk will touch upon many use cases where plac can make your life
as a developer simpler and better. plac works with all Python versions from 2.3
to 3.2 (more or less).
